Назначение программы
Данная программа предназначена для демонстрации работы криптосистемы RSA.

Она позволяет:

сгенерировать пару ключей RSA;
сохранить открытый и закрытый ключи в файлы;
зашифровать содержимое файла с использованием открытого ключа;
расшифровать зашифрованный файл с использованием закрытого ключа.
Программа реализована на языке Python без использования специализированных криптографических библиотек, что соответствует учебным целям практической работы.

Возможности программы
Программа поддерживает следующие операции:

Генерация ключей

случайный выбор двух простых чисел;
вычисление модуля n;
вычисление функции Эйлера φ(n);
выбор открытой экспоненты e;
вычисление закрытой экспоненты d.
Шифрование файла

чтение исходного файла в бинарном виде;
побайтное шифрование содержимого;
сохранение шифртекста в выходной файл в виде последовательности чисел.
Расшифрование файла

чтение зашифрованных чисел из файла;
побайтное восстановление исходных данных;
запись результата в новый файл.
Требования для запуска
Для работы программы необходимо:

наличие установленного Python 3.8 или выше;
наличие прав на чтение и запись файлов в рабочей директории.
Дополнительные библиотеки устанавливать не требуется, так как используются только стандартные модули Python.
